# FlickrUi

## Get started

To run this application, [Node](https://nodejs.org/en/download/) ([10.13.x](https://nodejs.org/dist/v10.13.0/)/[12.11.x](https://nodejs.org/dist/v12.11.1/) or later) must be installed. Additionally, it may be useful to have Angular CLI and Typescript installed globally.

Once Node is installed run `npm install` at the root of this repository, this will download the third-party dependencies.

To run the application locally, simply use `npm start`. Once running, the application will be available at `http://localhost:4200/`.
